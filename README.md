# Industrial Training Kit - Task 1

This project contains solutions and answers for Task 1 of the Industrial Training Kit using Python and Data Analysis libraries.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Files
- `industrial-training-kit2 (task1)Answers.ipynb`

## Features
- Data Cleaning
- Data Analysis
- Data Visualization
- Working with datasets using Python

## How to Run
1. Install the required libraries:
```bash
pip install -r requirements.txt
```

2. Open Jupyter Notebook:
```bash
jupyter notebook
```

3. Run the notebook file.
